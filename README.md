About
--

Microservice to get some system status info in JSON.

Available nodes:
--

- Uptime
- Load
- CPU (number of cores)
- Memory usage
- HDD

Installation:
--
`pip install system-status-server`

Usage
--
CLI for hdd stat:
```bash
hdd-stat
```

uWSGI example config:
```ini
[uwsgi]
processes = 2
module = system_status_server.app:app
http-socket = :35280
```
